# 简历定制 + 打招呼语

## Boss 打招呼语（初次触达）
- 初次"立即沟通"不必逐岗定制：设 1-3 条按岗位线的默认开场白，快速走量。
- 逐岗微调只在重点目标做。真正的定制发生在 HR 回复之后。
- 结构：一句定位（2.5年美股上市公司IR）+ 一个匹配点/成果数字 + 一个赛道钩子 + "方便聊聊吗"。≤4句。
- 只用简历能站住的事实，不编造。

## 三条基线模板（按岗位线）
1. 公司IR/证券事务/董秘：美股IR全流程 + 卖方研究 + SOX审计底色。
2. 一级市场IR/募资/投后：上述 + 信达机构销售(懂LP诉求) + 想深耕新兴赛道。
3. 港股/外资：强调华威硕士/雅思7.5/英文可作工作语言 + 美股IR迁移 + 在补港股规则。
按赛道把开头钩子换成 医疗/半导体/机器人 等。

## 简历定制（重排序不重写）
关键词对齐JD(严禁编造)、亮点前置、成果按岗位强调不同侧面、顶部摘要针对岗位重写。
