# 真实岗位评估：打分 + 硬性预筛（核心标准）

## 双维度打分（各 1-5）
- 匹配度：能力/经历与 JD 的重合度。
- 胜算：综合竞争、硬门槛、岗位真实度后的拿面试概率。
两者分开报（匹配高≠胜算高）。

## 分层
- 优先投(pri)：层级对口 + 赛道好 + 胜算高。
- 可冲(chong)：平台/赛道好但偏难，需差异化叙事。
- 补充(check)：层级合、信息待核。

## 硬性预筛（自动排除，除非平台极佳且候选人明确要冲）
1. 经验要求 > 候选人年限档（默认排除"5-10年""10年以上"；保留 经验不限/1年以内/1-3年/3-5年）。
2. JD/标题明确要求候选人不具备的硬条件：如 "FA/PE/VC 背景""美元基金资源""保荐代表人""X 年特定经验""募资经历/自带 LP 资源/政府募资关系"。
   - 募资岗区分：基金 GP 侧的募资岗(募资总监/合伙人/VP、政府募资、要求自带资源)→ 排除;公司侧投融资岗(实业公司为自身融资,考的是股权故事/材料/协调)→ 保留。一级市场优先「投后管理 + IR」。
3. 已投递/已联系过的公司（去重，避免重复打扰同一 HR）。
4. 赛道/地点明显错位、或岗位是纯投资/交易 deal 岗而候选人是 IR-track。

## 关键词配方（Boss 搜索）
好用：投资者关系 / IR / 投资者关系经理 / 证券事务代表 / 董事会秘书 / 投后管理 / 募资 / 投融资 / ECM。
避免单搜：资本市场(≈律所律师)、战略投资(≈投行/PE deal 岗，与 IR-track 错位)。
赛道叠加词：创新药/生物医药/医疗器械/CXO；半导体/芯片/集成电路/GPU；机器人/具身智能/智能驾驶。
